Немного истории JS

  • Был создан Бренданом Эйхом в 1995 г.
  • Развитие Single Page Application (SPA) ~2009
  • Примерно в это же время создается Node.js
  • Создание новой спецификации ECMAScript 6

Single Page Applications (SPA)

  • AJAX
  • Способ общения с сервером (почти всегда REST)
  • Роутинг

Проблемы: js, медленный старт, медленно на телефонах, часто не "прогрессивно", устаревание данных на клиенте, seo оптимизация...

Node.js

Node package manager (NPM)

  • в прошлом - пакетный менеджер для сервера
  • нет плагинов, но можно писать скрипты
  • package.json (как pom.xml)

Обзор других тулов (Awesome JS)

  • Пакетные менеджеры (npm, bower, jspm, ...)
  • Загрузчики (require.js, browserify, ...)
  • Сборщики (browserify, webpack, rollup, ...)
  • Транспайлеры (babel, typescript, ...)
  • Тестирование
    • Ранеры (phantomjs, karma, nightwatch, ...)
    • Фремворки (mocha, jasmine, jest, ...)
